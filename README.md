# Pong - Rust SDL2 Example

## Setup

You will need [rust](http://www.rust-lang.org/) and [SDL2](http://www.libsdl.org) on your system.

To build the examples via [Cargo](http://crates.io/)

1. `cargo build`

## [pong](src/pong.rs)

Play a game of pong
